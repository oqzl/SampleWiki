# ゲームモード

ゲームモードは、同じ基本ルールや素材を使いながら、目的、参加人数、制約、進行速度などを変えて異なる遊び方を提供する枠組みである。

## 設計上の役割

ゲームモードを分けることで、通常ルールを複雑化させずに、短時間向け、協力向け、競争向け、継続プレイ向けなどの体験を用意できる。

一方で、モードごとに専用ルールを増やしすぎると、学習負荷と調整コストが高くなる。基本メカニクスを共有し、勝利条件や初期配置など限られた差分で特徴を出す設計が扱いやすい。

## ローグライク的な利用

[[ローグライクSNS]]や[[ローグライクWiki]]では、探索重視、共同編集重視、期間限定イベントなどをモードとして切り替えられる。[[シーズン制]]と組み合わせる場合は、恒久的な知識や関係を失わせず、期間ごとの目標だけを変更する必要がある。
