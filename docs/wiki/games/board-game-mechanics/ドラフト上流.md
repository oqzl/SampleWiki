# ドラフト上流

ドラフト上流は、パスドラフトにおいて自分へ候補を渡してくる側のプレイヤー、またはその位置関係を指す。

上流側の選択は、自分が受け取れる候補の質と偏りを直接変える。したがって、自分の手元だけを見るのではなく、上流側が何を集め、何を残しやすいかを読むことが重要になる。

## 情報の読み方

上流側が特定の種類を継続して取っていると、その種類は自分まで流れてきにくい。この偏りを早期に察知できれば、競合の少ない方針へ切り替えられる。

一方、上流側の選択を過度に推測すると、偶然の候補配分を意図的な信号と誤認することがある。判断には複数巡の選択傾向や、公開された獲得物を組み合わせる必要がある。

## 設計上の意味

上流側から安定して有力候補が流れる状態は、プレイヤー間に暗黙の依存関係を生む。反対に、上流側が自分の欲しい候補を継続して取る場合は、競合関係が明確になる。

この関係をゲーム上の判断へ強く組み込むには、[[上流シグナル]]、[[競合回避]]、[[選択傾向推定]]などの情報を適度に観測可能にする必要がある。

方向が反転すると、上流側だったプレイヤーが下流側へ移る。そのため、ラウンドをまたいだ[[隣接関係反転]]が戦略へ与える影響も大きい。
