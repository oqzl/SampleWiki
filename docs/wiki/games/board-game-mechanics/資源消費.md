# 資源消費

資源消費は、行動、能力、移動、情報取得などの実行に有限の資源を支払わせるメカニクスである。

## 設計上の役割

すべての行動を同時には選べない状態を作り、何に資源を使うかを判断させる。資源が少なすぎると行動不能になり、多すぎると制約として機能しないため、補充速度と支出先の幅を合わせて調整する。

消費する資源は、通貨、時間、体力、カード、手番、権限など、ゲームの文脈に応じて置き換えられる。

## 探索への応用

[[ローグライクWiki]]では、ページを開く、リンクを有効化する、注釈を書くといった行為へ資源を要求することで、[[探索状態]]に選択を生める。ただし通常の情報閲覧を妨げないよう、ゲーム的なモードと分離する必要がある。
