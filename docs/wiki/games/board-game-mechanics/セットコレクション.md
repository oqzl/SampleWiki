# セットコレクション

セットコレクションは、特定の組み合わせや同種の要素を集め、その完成度に応じて得点や能力を得るメカニクスである。

## 基本構造

プレイヤーはカード、タイル、資源などを獲得し、同じ種類をそろえる、異なる種類を一式集める、指定された組み合わせを完成させるといった目標を追う。

得点方式には、集めるほど価値が増える方式、一定数で完成する方式、種類数を広げる方式などがある。

## 設計上の特徴

必要な要素が他のプレイヤーと重なると競争が生まれる。公開情報が多い場合は相手の狙いを妨害しやすく、非公開の場合は確率や残数の推測が中心になる。

[[ドラフト]]と組み合わせると、自分のセットを伸ばす選択と、相手へ必要な要素を渡さない選択が同時に発生する。
