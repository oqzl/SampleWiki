# ボードゲームのメカニクス

ボードゲームにおけるメカニクスとは、プレイヤーが何を選び、どのように状態を変化させ、勝敗へ近づくかを規定する仕組みである。

## 設計上の観点

メカニクスは単独で面白さを生むというより、他の仕組みとの組み合わせによって体験を作る。

たとえばドラフトは、カードを選ぶだけの仕組みであるが、選ばなかったカードが隣へ渡ることで、情報戦や妨害の意味が生まれる。同じメカニクスでも人数、公開情報の量、得点方式によって感触は大きく変わる。

また、[[フレーバーとメカニクス]]が一致すると、プレイヤーはルールを暗記するのではなく、世界の因果として理解しやすくなる。
