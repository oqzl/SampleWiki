# 隠匿情報

隠匿情報は、一部の情報を特定のプレイヤーだけが知る状態にし、推測、読み合い、伝達制限を生むメカニクスである。

## 基本構造

手札、個別目標、正体、伏せられた配置などを非公開にすることで、各プレイヤーが異なる情報を持つ。

完全に秘密にするだけでなく、情報の一部だけを公開する、後から公開する、行動によって推測可能にするといった段階的な設計も使われる。

## 設計上の特徴

情報が少なすぎると当てずっぽうになり、多すぎると隠す意味が薄れる。公開情報から合理的に推測できる余地を残すことが重要である。

[[協力型ゲーム]]では、全員が同じ情報を持たないことで、一人の熟練者がすべての判断を支配する問題を弱められる。[[ドラフト]]では、獲得物や候補の流れをどこまで公開するかによって読み合いの強さが変わる。
