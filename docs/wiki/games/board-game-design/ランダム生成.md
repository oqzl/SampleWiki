# ランダム生成

ランダム生成は、ゲーム開始時や進行中に、盤面、候補、イベント、経路などを一定の規則に従って変化させる設計手法である。

完全な無作為ではなく、出現範囲、頻度、組み合わせ禁止、最低保証などの制約を設けることで、予測不能さと遊びやすさを両立させる。

## 主な用途

- 毎回異なる初期配置を作る
- 選択可能な行動や報酬を入れ替える
- イベントの順序を変える
- 探索経路や遭遇内容を変化させる

## 設計上の注意

結果の振れ幅が大きすぎると、判断より運が勝敗を支配する。プレイヤーが結果へ対応できる余地、悪い結果から回復する手段、偏りを抑える仕組みが必要である。

ランダム生成はリプレイ性を高めるが、選択の意味を自動的に生むわけではない。生成された状況を読み、方針を変える判断と組み合わせて初めてメカニクスとして機能する。
