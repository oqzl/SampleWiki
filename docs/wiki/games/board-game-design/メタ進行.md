# メタ進行

メタ進行は、一回のプレイや一つのランが終了した後も、次回へ影響する要素を残す進行構造である。

その場限りの得点や資源とは異なり、解放された機能、発見済み情報、開始条件の変化などが複数回の体験をつなぐ。

## 主な形

- 新しい選択肢や開始地点の解放
- ルールや情報の段階的な公開
- 過去の結果を反映した初期条件
- プレイヤー固有の記録や収集物

強化だけを恒久的に積み上げると、後から参加した人との差や作業感が大きくなりやすい。数値上の優位だけでなく、選択肢、知識、表現、探索経路を増やす方向にも設計できる。

## 一回のプレイとの関係

各ランを独立した判断の場として保ちながら、複数回を通した目的を与える。[[重大な選択]]や[[ランダム生成]]と組み合わせる場合、今回だけの成果と次回へ残す成果を分けることで、短期と長期の優先順位が生まれる。
