# ランダムイベント

ランダムイベントは、ゲーム中の一定条件や抽選によって発生し、盤面、選択肢、資源、関係などを一時的に変化させる出来事である。

## 役割

毎回同じ手順になることを防ぎ、状況への対応を促す。単なる不運ではなく、発生後に複数の対応策があると、予測不能さが判断へ変わる。

イベントの影響が大きすぎると、それ以前の選択を無効化しやすい。発生頻度、影響範囲、事前予告、回避手段を調整し、プレイヤーが結果へ関与できる余地を残す必要がある。

## WikiやSNSへの応用

WikiやSNSでは、期間限定の表示、共同編集の機会、特定ページへの入口などをイベントとして扱える。本文そのものを壊さず、表示経路や一時権限だけを変える方が、共同知識を維持しやすい。
