# 永続レリック

永続レリックは、一度のプレイや探索が終了した後も保持され、次回以降の選択肢や見え方を変える要素である。

## 役割

単純な数値強化ではなく、新しい入口、別の表示方法、再抽選、追加の選択肢などを解放すると、周回後も判断の幅を広げられる。

永続効果が強すぎると初回利用者との差が固定されるため、勝敗を直接決める能力より、探索方法や情報への接し方を変える効果が扱いやすい。

## Wikiへの応用

[[ローグライクWiki]]では、特定分野への入口、バックリンクの表示、未訪問ページの示唆などをレリックとして扱える。共同のページ本文は変えず、利用者ごとの[[表示経路]]だけを拡張する。
