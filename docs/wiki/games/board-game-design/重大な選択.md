# 重大な選択

重大な選択は、その後の進行、利用可能な選択肢、得点機会、他者との関係などを大きく変える判断である。

単に数値上の最善手を選ぶのではなく、何かを得る代わりに別の可能性を失う点に特徴がある。選択後に簡単には元へ戻れないほど、その判断は重く感じられる。

## 設計上の役割

重大な選択には、主に次の役割がある。

- プレイヤーごとの展開を分岐させる
- 方針や役割を明確にする
- 短期的利益と長期的利益を対立させる
- 選択結果への納得感や物語性を生む

選択肢の差が見えなさすぎると運任せに感じられ、結果が完全に予測できると計算問題になりやすい。判断材料を与えつつ、将来の不確実性を適度に残す必要がある。

## ローグライク構造との関係

一度のランで得られる資源や経路が限られる場合、どの選択肢を捨てるかがラン固有の特徴になる。[[ランダム生成]]や[[変則的な勝利条件]]と組み合わせることで、同じ状況でも異なる優先順位が生まれる。
