# ChatGPT連携

## 現在の方式

ChatGPTは通常リポジトリ内の `docs/wiki` を編集する。

GitHub Actionsが変更を検出し、`SampleWiki.wiki.git` へ同期する。

## 処理の流れ

1. ChatGPTが対象ページを読む
2. Markdownを更新する
3. `main` ブランチへコミットする
4. `Sync Wiki` ワークフローが起動する
5. GitHub Wikiへ反映する

## 依頼例

> WikiベースSNSのページに、ページフォロー機能の案を追記して。

> 運用ルールを確認して、曖昧な箇所を整理して。

> 新しいページ「ページの統合」を作り、関連ページからリンクして。

## 安全策

- 削除やページ統合は自動実行しない
- 大きな変更は差分を先に確認する
- 同じ依頼を再実行しても重複追記しない
- Wiki画面からの直接編集内容は同期で消える可能性がある
- 外部事実は調査と出典確認を行ってから記述する
- 本文は原則として「だ・である調」で統一する
