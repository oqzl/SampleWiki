# ビールの泡

ビールをグラスへ注いだとき、液面にできる泡。

## 雑学

キリンの公式FAQによると、ビールの泡は炭酸ガスの気泡に、ビール中のたんぱく質、炭水化物、ホップ樹脂などが付着して形成される。

泡には、炭酸ガスが逃げるのを抑え、ビールが空気に触れて味が落ちるのを防ぐ「蓋」の役割がある。サッポロビールも、泡が炭酸や香りの急な発散と空気接触による酸化を防ぎ、おいしさを保つと説明している。

サッポロビールのFAQでは、液体と泡の比率は7対3が目安とされている。ただし、注ぎ方によって泡の質や口当たりが変わるため、比率だけで味が決まるわけではない。

ホップ由来の成分は苦味と香りだけでなく、泡持ちにも関係する。したがって泡は、単なる炭酸の副産物ではなく、麦芽、ホップ、グラスの状態、注ぎ方が合流する場所とも言える。
