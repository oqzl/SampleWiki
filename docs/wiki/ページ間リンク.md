# ページ間リンク

このWikiで、あるページから別のページへ移動できるようにする接続である。

## このWikiでの役割

ページ間リンクは、文章を分類棚へ収納するためだけのものではない。本文中の特徴的な語から別の話題へ脱線し、予想外のページへ回遊するための入口でもある。

リンク元からリンク先へ進む通常のリンクに加え、リンク先からリンク元を確認する仕組みが[[バックリンク]]である。

## 書式

    [[ページ名]]

表示名とページ名を分ける場合は次の形式とする。

    [[表示名|ページ名]]
