# 招待制

招待制は、既存の参加者や運営者から招待された人だけが、新たにサービスや共同体へ参加できる方式である。

## 目的

参加者の増加速度を制御し、共同体の文化や安全性を保ちやすくする。誰が誰を招待したかを記録すれば、参加経路や責任範囲も追跡できる。

一方で、既存参加者との接点がない人を排除しやすく、閉鎖性や固定化を生む。招待枠を希少資源として扱う場合は、参加機会の公平性と悪用防止を同時に設計する必要がある。

## ゲーム的な利用

招待権を獲得条件や報酬にすると、参加者の行動に意味を持たせられる。ただし、招待者へ過剰な連帯責任を課すと、人間関係の監視や萎縮につながる。
