# 招待枠

招待枠は、利用者が新しい参加者を招待できる回数または人数の上限である。

## 目的

招待制のサービスで参加者の増加速度を制御し、無差別な招待や招待コードの大量配布を抑える。枠を有限にすると、誰を招待するかが選択になる。

一方で、招待枠を希少にしすぎると、既存参加者との接点がない人が参加しにくくなる。一定期間で回復させる、活動状況に応じて追加する、運営用の別経路を用意するなど、公平性を補う設計が必要である。

## ゲーム的な扱い

招待枠を資源として扱う場合も、人気や課金だけで恒久的に増え続ける構造は避ける。利用者間の格差を固定せず、共同体への貢献や適切な運用と結び付ける方がよい。
