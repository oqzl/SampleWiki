# 一時権限

一時権限は、利用者やグループへ一定期間または特定条件の間だけ付与される操作権限である。

## 用途

常設の管理者権限を増やさずに、イベント運営、レビュー、限定編集、障害対応などへ必要な操作を委ねられる。

期限、対象ページ、可能な操作を明確に限定すると、権限の戻し忘れや過剰付与を抑えやすい。期間だけでなく、作業完了、イベント終了、承認取消などを失効条件にできる。

## Wikiでの扱い

[[WikiベースSNS]]や[[ローグライクWiki]]では、特定ページ群の編集、[[共同イベント]]の進行、公開前確認などへ利用できる。[[招待制]]と組み合わせる場合も、参加資格と編集権限を分けて管理する必要がある。
