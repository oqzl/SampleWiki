# 知識空間

知識空間は、ページ、概念、リンク、履歴などを、探索可能な一つの構造として捉える考え方である。

## Wikiにおける構造

Wikiでは、ページ単体よりも、[[ページ間リンク]]と[[バックリンク]]によって形成される関係の網が重要になる。利用者は検索だけでなく、ある概念から近接する別の概念へ移動しながら理解を広げる。

同じページ集合でも、入口、表示順、既知の概念、編集履歴によって見える範囲は変わる。そのため知識空間は固定された目次ではなく、利用者ごとに異なる探索体験を生む構造でもある。

## 設計上の観点

分類は全体の位置を把握しやすくし、リンクは分類を越えた意味関係を示す。両者を併用することで、整理と偶発的な発見を両立できる。
