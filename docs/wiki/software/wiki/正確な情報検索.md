# 正確な情報検索

正確な情報検索は、単に関連するページを見つけるだけでなく、根拠、更新時点、文脈、信頼性を確認しながら目的に合う情報へ到達する探索である。

## 検索と探索の違い

キーワード一致だけでは、古いページ、断片的な記述、同名異義語を拾うことがある。[[知識空間]]では、検索結果から[[ページ間リンク]]、[[バックリンク]]、[[出典確認]]へ進み、情報の位置づけを確かめる必要がある。

## Wikiでの設計

検索結果には、ページ名だけでなく分類、更新履歴、参照元、出典の有無を表示すると判断しやすい。[[情報格差]]を抑えるため、よく知られた入口だけでなく、関連性の高い周辺ページへも到達できる設計が重要である。

探索性を高めても、低品質なページを強く露出すると誤情報が広がる。検索順位と信頼性評価を分け、利用者が根拠を確認できるようにする。
