# 出典確認

出典確認は、ページに記載する事実について、根拠となる資料の内容、発行主体、更新時期、適用範囲を確かめる作業である。

## 確認項目

- 一次資料または公式情報か
- 記載内容が現在も有効か
- 引用部分が文脈と一致しているか
- 商品説明と第三者検証を混同していないか
- 推測、本人談、確認済み事実を区別しているか

単にURLを付けるだけでは、根拠として十分とは限らない。リンク先が主張を直接支えているかを確認し、情報が変わり得る場合は確認日を残す。

## Wiki運用との関係

[[編集履歴]]と組み合わせることで、どの時点の資料に基づいて記述したかを追跡できる。内容が競合する場合は、資料の信頼性と対象範囲を示し、断定を避けて併記する。

[[共同知識]]を維持するには、出典の追加だけでなく、リンク切れや古い根拠の定期的な見直しも必要である。
