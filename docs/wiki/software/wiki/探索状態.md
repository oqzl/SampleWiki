# 探索状態

探索状態は、利用者が知識空間のどこまで到達し、何を発見し、現在どの条件で閲覧しているかを表す状態である。

## 構成要素

探索状態には、現在位置、訪問済みページ、発見済み概念、利用可能な入口、一時的な権限などを含められる。

ページ本文や[[編集履歴]]のような共同情報とは分離し、利用者ごとの表示や進行だけを保持すると、Wikiの正本を壊さずに探索体験を変えられる。

## リセットと持ち越し

[[ローグライクWiki]]や[[ループ系]]では、一部の探索状態をラン終了時に初期化し、[[発見済み概念]]など必要な情報だけを持ち越す。何を失い、何を残すかが体験の設計になる。
