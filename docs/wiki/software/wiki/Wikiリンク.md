# Wikiリンク

Wikiリンクは、ページ名を指定してWiki内の別ページへ接続するリンク記法である。

SampleWikiでは `[[ページ名]]` を基本形とし、表示名を変える場合は `[[表示名|ページ名]]` を使う。

## 役割

通常のURLを直接書く方式と異なり、ページ名を知識の識別子として扱える。リンク先がまだ存在しない場合でも記述できるため、後からページを作成して知識ネットワークを拡張できる。

一方、未作成リンクを無制限に増やすとWikiが収束しない。下位概念や同義語は独立ページにせず、既存ページの節へ統合する判断も必要である。
