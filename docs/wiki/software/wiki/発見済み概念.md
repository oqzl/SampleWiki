# 発見済み概念

発見済み概念は、利用者が探索や閲覧を通じて一度到達し、その後の表示や進行で既知として扱われる概念である。

## 役割

未訪問の概念と既知の概念を区別すると、同じWikiでも利用者ごとに探索履歴を持たせられる。既知の概念は再訪しやすくし、未訪問の概念は存在だけを示すなど、表示段階を分けられる。

## 永続化

ページ本文や[[共同知識]]とは分離し、利用者ごとの状態として保持する。本文の正しさや公開範囲を変えずに、[[表示経路]]や推薦順だけを変化させられる。

発見状態を永続化しすぎると初見の探索感が失われるため、[[ループ系]]では一部だけを持ち越す、分野単位で再発見を要求するなどの調整が考えられる。
