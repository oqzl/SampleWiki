# 共同イベント

共同イベントは、複数の利用者が同じ期間、目標、話題、ページ群に参加し、成果を持ち寄る企画である。

## Wikiでの構造

Wikiでは、単一の投稿欄へ集約するのではなく、参加者が別々のページを作成し、[[ページ間リンク]]と[[バックリンク]]によって成果を接続できる。

[[アドベントカレンダー]]のように日付で担当を分ける方式、共通テーマから枝分かれする方式、既存ページを順番に改善する方式などが考えられる。

## 設計上の要点

- 参加期間と完了条件を明示する
- 新規作成と既存ページ更新の両方を許可する
- 重複ページを統合できる導線を用意する
- イベント終了後も成果へ到達できる状態を保つ

一時的な盛り上がりだけで終わらせず、[[共同知識]]として残る構造にすると、次回の企画や日常の探索へ接続しやすい。
