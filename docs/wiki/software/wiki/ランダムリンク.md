# ランダムリンク

ランダムリンクは、固定された関連ページ一覧ではなく、候補集合から一定の規則で選ばれたリンクを表示する仕組みである。

## 役割

毎回同じ経路をたどる状態を避け、既存ページから予想外の概念へ移動できるようにする。完全な無作為にすると文脈を失いやすいため、現在のページとの関連度、未訪問状態、閲覧履歴、カテゴリなどを重みとして使う。

## Wikiでの利用

[[ローグライクWiki]]では、ランごとに異なる[[表示経路]]を作る基礎になる。再抽選回数や候補範囲を[[有限資源]]として扱うと、どのリンクを残すかが選択になる。
