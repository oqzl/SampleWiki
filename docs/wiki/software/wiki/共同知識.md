# 共同知識

共同知識は、複数の利用者が参照し、追記し、修正できる形で蓄積された情報である。

個人の記憶や一時的な投稿と異なり、後から参加した人も同じ情報へ到達でき、変更の経緯や関連概念をたどれることが重要になる。

## Wikiでの構成要素

- ページ本文
- [[編集履歴]]
- 出典
- [[ページ間リンク]]
- [[バックリンク]]

共同知識を維持するには、誰でも変更できることだけでなく、重複、表記揺れ、誤情報、過細分化を整理する仕組みが必要になる。

## 個別体験との分離

閲覧順、推薦結果、一時的な権限などは利用者ごとに変化させても、ページ本文や履歴は共有状態として残せる。この分離により、探索体験を変化させながら知識そのものの蓄積性を保てる。
