# 低品質編集

低品質編集は、本文量、根拠、接続性、重複確認が不足し、Wiki全体の探索性や信頼性を下げる編集である。

## 主な兆候

- 一文だけで主題を説明できていない
- 既存ページと重複している
- 根拠が必要な事実に出典がない
- 文脈の薄い未作成リンクを大量に増やす
- 報酬や件数達成だけを目的にページを作る

## 防止方法

新規ページを作る前に、既存ページへの統合可能性と[[バックリンク]]を確認する。外部事実を扱う場合は[[出典確認]]を行い、公開前の内容は[[下書き領域]]で整理する。

自動生成やゲーム的な報酬を導入する場合は、作成数ではなく、未作成リンクの純減、既存ページとの接続、重複回避を評価対象にする。
