# ページ名

ページ名は、Wiki内で一つの話題を識別するための名称である。SampleWikiではMarkdownファイルの見出しとファイル名を対応させ、Wikiリンクの接続先として使う。

## 命名方針

- 内容を端的に表す名詞または複合語を使う
- 同義語による重複ページを避ける
- 文脈がなくても意味が分かる名称を優先する
- 小見出しで十分な下位概念は独立ページにしない

曖昧な名称は、上位概念や対象分野を含む複合語にすると識別しやすい。ただし細分化しすぎるとページと未作成リンクが増えるため、検索性と統合可能性の釣り合いを取る。

## 表示名

本文中で自然な表現を使いたい場合は、`[[表示名|ページ名]]` の形式で表示文字列とリンク先を分ける。
