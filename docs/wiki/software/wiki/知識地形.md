# 知識地形

知識地形は、ページとリンクの集合を、入口、近接関係、到達しやすさ、未訪問領域を持つ探索空間として捉えた表現である。

## 構成要素

同じ[[知識空間]]でも、どのページを入口にするか、どのリンクを強調するか、どこまで閲覧済みかによって利用者から見える地形は変わる。

分類は大まかな地域を示し、[[ページ間リンク]]と[[バックリンク]]は地域間の経路になる。未訪問ページを直接表示せず、存在だけを示す方法も探索感を作る。

## ローグライクWikiでの利用

[[ローグライクWiki]]では、ページ本文を変更せず、[[表示経路]]や[[ランダムリンク]]だけを変えて周回ごとに異なる知識地形を作れる。
