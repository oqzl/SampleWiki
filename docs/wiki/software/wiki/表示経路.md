# 表示経路

表示経路は、利用者がどのページや情報を、どの順序で提示されるかを表す経路である。

同じページ群を共有していても、入口、推薦、閲覧履歴、権限、ランダム性によって表示経路は変化する。知識そのものではなく、知識へ到達する順序を設計対象として扱う考え方である。

## 主な決定要因

- 最初に表示される入口ページ
- [[ページ間リンク]]の選択
- [[バックリンク]]や関連ページ
- 推薦やランダム提示
- 閲覧可能範囲や一時的な制限

表示経路を固定すると理解順序を制御しやすい。一方、経路を変化させると偶然の発見や再訪時の新鮮さが生まれるが、重要情報へ到達できない危険もある。

## 共同知識との分離

[[共同知識]]を全利用者で共有しつつ、表示経路だけを個別化できる。これにより、本文を利用者ごとに複製せず、探索体験だけを変化させられる。
