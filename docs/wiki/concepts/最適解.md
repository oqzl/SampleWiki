# 最適解

最適解は、与えられた条件と評価基準のもとで、最も高い結果を得られる選択や手順である。

## ゲームにおける意味

ゲーム中に常に同じ最適解が成立すると、プレイヤーの判断は発見後に自動化しやすい。反対に、情報が不完全であったり、他者の行動によって状況が変化したりすると、最適解は一つに定まりにくくなる。

設計では、唯一の正解を隠すだけでなく、複数の有力な方針、短期利益と長期利益の衝突、資源制約などを組み合わせることで判断の幅を作る。

## 探索との関係

最適解を探す行為そのものが面白さになる場合もある。ただし一度発見された解が共有され、以後の選択を支配する場合は、条件の変化や複数目標によって固定化を防ぐ必要がある。
