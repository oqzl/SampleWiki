# 投稿意欲

投稿意欲は、利用者が文章、画像、反応、編集などを新たに共有しようとする動機の強さである。

## 影響する要素

投稿までの手間、反応の得やすさ、失敗への不安、公開範囲、内容が後から役立つ感覚などによって変化する。

報酬だけを強くすると、量を増やすための低品質な投稿を誘発しやすい。反対に、喪失や評価低下のリスクが大きすぎると、閲覧だけを選ぶ利用者が増える。

## Wiki型サービスでの扱い

ページや概念へ投稿を蓄積する場合、過去の内容が再発見されること自体が動機になり得る。下書き領域、一時的な公開範囲、編集履歴などを用意すると、完成度への不安を抑えやすい。
