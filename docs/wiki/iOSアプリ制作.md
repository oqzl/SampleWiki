# iOSアプリ制作

iPhoneやiPad向けのアプリを設計・実装すること。

## 最近試している題材

- カメラと画像処理
- Markdown編集とGitHub同期
- 楽譜閲覧と書き込み
- 騒音計やセンサー利用
- 画像ビューアーと独自ジェスチャー
- ジョークアプリや実験的UI

## 雑学

iOSアプリは、見た目の実装だけでなく、バックグラウンド実行、権限、端末回転、カメラ制御、App Store審査など、OS側の制約を前提に設計する必要がある。

そのため、単純に「機能が実装できるか」だけでなく、「ユーザーが自然に許可できるか」「アプリが停止した後も成立するか」「審査上問題がないか」まで含めて設計対象になる。
