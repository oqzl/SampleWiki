# 運用ルール

## 運用主体

このWikiは、主にChatGPTが会話内容や既存ページをもとに、かなり勝手にページを作成・追記・リンクします。

人間は方針の指定、事実確認、削除や統合などの大きな判断を担当します。内容には雑学的な脱線や未整理のアイデアを含めてよいものの、外部世界に関する事実を推測で埋めてはいけません。

## 調査と出典

- 商品、人物、企業、歴史、科学、医療、制度、仕様、価格などの外部事実は、書く前に必ず調査する
- 可能な限り、公式サイト、公的機関、一次資料、公式ドキュメントを優先する
- 調査していない内容を、もっともらしい一般論で補完しない
- 確認できなかった内容は断定せず、「未確認」「本人談」「推測」などの区別を明記する
- 医療・健康・法律・金融に関する内容は、商品説明と科学的根拠を混同しない
- 雑学ページにも、原則として「出典」節を設ける
- 情報が後から変わり得る場合は、必要に応じて確認日を記載する

## 原本

Wikiページの原本は `docs/wiki/*.md` です。

GitHub上のWiki画面から直接編集すると、次回同期時に上書きされる可能性があります。通常はリポジトリ側のMarkdownを編集します。

## ページ名

- 1ページにつき1つの主題を扱う
- 同じ意味のページを重複して作らない
- ページ名は短く、検索しやすい表現にする
- 人名ページは本人の同意と管理方針を確認してから作る

## 変更

- ChatGPTは既存ページから特徴的な単語を拾い、リンク先の雑学ページを追加してよい
- ランダムリンク先の雑学も、調査と出典確認を行ってから作る
- 大幅な削除や統合は、変更理由をコミットメッセージに残す
- ページ名変更時はリンク元も更新する
- 誤りを見つけた場合は、黙って残さず修正し、必要なら修正理由を記録する

## プライバシー

- 趣味、制作テーマ、公開してよいと本人が指定した嗜好は記載してよい
- 家族、住所、勤務先、収入、健康状態など、私生活に踏み込む情報は本人の明示的な許可なく書かない

## リンク

GitHub Wiki形式のリンクを使用します。

    [[ページ名]]

表示名を変える場合は次の形式です。

    [[表示名|ページ名]]
